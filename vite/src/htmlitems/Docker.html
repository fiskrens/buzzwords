Docker

Verktyg för att skapa upp virtuella servrar baserat på konfig-filer. Används ofta i kombination med docker-compose.

Med hjälp av docker-compose kan man skapa en fil som innehåller instruktioner om hur själva server-miljön ser ut. Docker kan sedan virtuellisera upp hela denna miljön med de versioner som är specificerade.

Detta gör att release av nya versioner, uppskapning av miljöer lokalt vid utveckling samt risken till problem med konflikter i kod kontra server-miljö minskar.
