JSON

Ett standardiserat läsbart format på data som används för att lagra eller skicka data mellan olika system.

Formatet lagras med attribut: värde, alltså ett värde per attribut. Attribut kan dock innehålla flera attribut, eller en array med flera värden.

Exempelkod

{
  "first_name": "Anders",
  "last_name": "Larsson",
  "age": 36,
  "address": {
    "street_address": "Storgatan 1",
    "city": "Växjö",
    "state": "Kronoberg",
    "postal_code": 35232
  },
  "frogs": [
    "Bufo punctatus", 
    "Rana temporaria",
    "Epidalea calamita"
  ]
}
